# Aggie2Aggie Connect

A modern engagement platform for Texas A&M University students, alumni, and faculty. Built with React, TypeScript, and Tailwind CSS using official Texas A&M brand colors.

## Brand Colors

This project uses the official Texas A&M University brand colors:

- **Aggie Maroon**: `#500000` (Primary brand color)
- **Aggie Dark Maroon**: `#3C0000`
- **Aggie Ochre**: `#D6D3C4`
- **Aggie Navy**: `#2F3E51`
- **Aggie Gray**: `#707070`
- **Aggie Gray Light**: `#D1D1D1`

## Features

- **Landing Page**: Modern, glass-morphism design with hero section, features, and CTA
- **Authentication**: Login and signup views with form validation
- **Responsive Design**: Mobile-first approach with hamburger menu
- **Community Features**: Mentorship, events, and resource discovery
- **Growth Analytics**: Track engagement and development

## Tech Stack

- React 18.2
- TypeScript 5.2
- Vite 5.0
- Tailwind CSS 3.3
- Lucide React (Icons)

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <your-repo-url>
cd A2AConnectSample
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser to `http://localhost:5173`

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
